<h1>Portofolio</h1>

<p>Website ini merupakan website Portofolio portfolio pribadi yang digunakan untuk menampilkan informasi project dan skill saya.Website ini memiliki halaman utama yang berisi: Deskripsi Singkat, latar belakang dan kemampuan developer. Selanjutnya ada pengalaman proyek yang pernah di kerjakan oleh developer setiap proyek dilengkapi dengan teknologi yang digunakan. Pada bagian akhir website terdapat informasi kontak yang memungkinkan pengunjung untuk menghubungi developer apabila memiliki proyek atau kerja sama yang ingin dilakukan.</p>
